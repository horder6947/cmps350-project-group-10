## Project Overview

This is a front-end social media application created entirely using HTML, CSS, and JavaScript. The application is designed to store its data in local storage within the browser. The application is equipped with major features such as user registration/login, dynamic feed functionality, posting, deleting of posts, liking, commenting, profiles, and following/unfollowing.

As for the backend of the application, there is no server or database used in this application. Instead, it is created using a simulated backend using JavaScript modules. The application is created in such a way that it is completely independent as a static web application.

## How to Use the Website

### Logging into an Existing Account

- Enter the email and password.
- Click on **"Log In"**.
- If you don't have an account or want to create one, click on **"Register Here"**.

### Registration of a New User

- Navigate to the registration page.
- Enter your name, email, and create a password.
- Click on **"Create Account"**.

### Following / Unfollowing Accounts

- Navigate to the **Discover** tab to discover users.
- You can see all the users that have an account along with their followers, following, and post count.
- Click on **Follow** / **Unfollow** to follow or unfollow users.

### Viewing and Interacting with Posts

- In the **Feed** tab, you can see the posts of the users you follow.
- You can like or comment on the posts.

### Uploading a New Post

- Go to the **New Post** section to make your own post.
- Click on the _"What do you want to share?"_ text and put your content.
- Press **"Publish Post"** to successfully upload your post.

### Viewing Your Profile

- View your profile by going into the **Profile** tab.
- Click on **"Edit Profile"** to edit your username and bio.
- Select **"View Feed"** to go back to your feed.

### Switching Between Light / Dark Modes

- Toggle light mode on/off by clicking on the icon to the left of the **Discover** tab.
